Analytics Builder User Guide



The Analytics Builder provides a graphical interface which is used to configure the analytics which are supplied with the Industry Analytics Kit.
   
   

The builder consists of five main areas :

  • Navigation Panel
          The navigation bar at the top of the screen provides common functionality such as clearing the configuration, loading saved configurations, saving new configurations and loading sample configurations.

  • Analytics Panel
            The analytics panel on the left of the screen displays the list of available analytics. These analytics have been categorised into groups according to their function.

  • Configuration Panel
            In the centre of the screen is the main configuration panel. This is the main work area which is used to create a new configuration or edit a previously saved configuration.

            The configuration panel allows a user to drag in analytics, link them by channels and change the properties of analytics

  • Documentation Panel
            At the bottom of the screen is the documentation panel. This panel displays the documentation for the analytic which has been selected either in the Analytics Panel or in the main Configuration Panel

  • Properties Panel
           On the right of the screen is the Properties Panel. This either displays the Channel Properties if a channel has been selected in the main Configuration Panel or the Analytic Properties if an analytic has been selected


Analytic Samples

    The Analytics Builder is provided with three samples which can be loaded from the 'Samples' dropdown in the navigation bar.

    Hovering over these samples displays their short description.When a sample is clicked, it is automatically loaded and displayed in the Configuration Panel.

    Missing Spikes
        This configuration consists of a Spike Analytic which is receiving data from a channel called RAW_SENSOR_DATA.
        The output from the Spike Analytic is being sent into the Missing Data Analytic which is sending the output to a SENSOR_ANOMALIES channel


    Peer Analysis
        The Average analytic receives raw data from the Orders channel which is then output to the Average channel.
        Both the raw Orders data and the Average which has been calculated by the Average analytic is fed into the Spread Analytic. The Spread analytic outputs the spread calculated from these two input streams.
        The Threshold analytic receives the output from the Spread analytic and outputs any data which is outside of the threshold (defined in its properties) to the Breaches channel.


    Volatility Increase
        The data from the RAW_SENSOR_DATA channel is received by the Volatility analytic which calculates the volatility using variance over a default 60 second time window.
        The output is sent to a Threshold analytic which is detecting data breaches with a value greater than or equal to 25. These breaches are output to the SENSOR_ANOMOLIES channel.


How to use the Analytics Builder

        The analytics supplied with the Industry Analytics Kit are listed under their relevant categories (Detectors, Extensions, Flow Manipulation, GeoLocation, Streaming Calculations, Utilities).

        The categories can be expanded by clicking on the category name.


        Adding an analytic to the Configuration Panel
        To add an analytic into the confguration panel, left click on the analytic in the Analytic Panel. Whilst holding down the left mouse button, drag and drop the the analytic over one of the grey drop areas in the Configuration Panel.


        Moving an analytic on the Configuration Panel
        To move an analytic from one location to another in the configuration panel, left click on the analytic and drag it to a new grey drop location.


        Copying an analytic
        To copy an analytic which is already in the Configuration Panel, left click to select the analytic. Whilst holding down the left mouse button, hold down the 'ctrl' button on the keyboard.
        When the analytic is dragged to a new grey drop location, it will be copied and not moved.


        Deleting an analytic from the Configuration Panel
            Left click on the analytic. Hold down the left mouse button and drag the analytic off the configuration panel.       
           
            Note: A faster way to delete an analytic is to left click on the analytic. When it has been selected , press the 'del' key on the keyboard to delete the analytic.

   
        Joining Analytics together
        When an Analytic is dragged into the Configuration panel, the default input and output channels are automatically linked to the analytics which are to the left and right.
        If the analytic contains any optional channels, these need to be connected manually by clicking on the small white circle on the analytic. 


        Changing Channel names
 
        When an analytic is dropped at the start of end of a row, the input channel name is defaulted to the input channel name defined for that analytic.
       
        When an analytic is dropped at the end of a row, the output channel name is defaulted to the output channel name defined for that analytic.

        These default input and output channels can be renamed.

        To select an input or output channel, hover the mouse over the channel name or blue circle in the configuration panel. When the mouse cursor changes to a hand pointer, left click to select.
       
        When the channel is selected, the right Properties Panel will display the 'Channel Properties' and the Channel Name property will be highlighted.

        Type in the Channel Name input field to change the name of the channel.

        Note: If you need to change more than one channel to the same channel name, update the first one, then select and drag the channel down onto the other channel locations
       

        Changing the properties on an analytic
       
        An analytic can contain multiple properties which can be changed as required.
       
        To change the properties of an analytic, select the analytic by left clicking on it in the Configuration Panel. When the analytic is selected, the 'Analytic Properties' will be displayed in the Properties Panel.

        To change the property, click on the required field in the Analytic Properties panel.


        Saving a configuration
   
        To save the configuration, click on the 'SAVE' option in the navigation bar. Enter a name and description and click on 'Save Changes'.


        Loading a configuration

        To load a previously saved configuration. click on the 'LOAD' option in the navigation bar. Select the .evt file and click on 'Open'. The file will be loaded in to the configuration panel.

        To see the description for the configuration, hover over the filename in the navigation bar.


        Clearing the current configuration

        To clear the current configuration from the configuration panel, click on the 'NEW' option in the menu bar. If you have unsaved changes, a dialog will display with a warning.


        Additional Features

  • Undo / Redo
            The configuration panel contains undo / redo functionality which ensures that you are able to reverse any changes you have made whilst in the configuration panel.

            The undo / redo will reverse any changes to the channel names, analytics drag and drop positions plus any analytic property changes.
